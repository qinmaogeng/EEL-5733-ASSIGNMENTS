How to compile this code: gcc -o test test.c -pthread      Or you can go into the path and type make
How to run the program:"  ./test 10 5  <input.txt   The first argument 10 represents the size of each buffer The second argument 5 represents the number of buffers and reducers.
The program will print the process and status of each thread and the result of each reducer on the screen.
The final result of every reducers is outputed to the output.txt
